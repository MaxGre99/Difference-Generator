## Description:
  Program, generating differences between two JSON or YAMl/YML files. Returns result in tree, plain or JSON formats.

## Installation:

  ### Installation
    make install

  ### Run Tests
    make test

  ### Lint
    make lint

  ### Quick Start 
    gendiff -h
